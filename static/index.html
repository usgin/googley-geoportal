USGIN Metadata Proxy Thing!

You can use this server to do some things!

Replace the [metadata indentifier] part of these URLs with the identifier for a record in the USGIN Metadata Catalog.

Or click one of the links for an example record.

  1. See a metadata record as XML: http://metadata.usgin.org/record/[metadata identifier]?f=xml
  2. See a metadata record as JSON: http://metadata.usgin.org/record/[metadata identifier]?f=json
  3. See a metadata record as HTML: http://metadata.usgin.org/record/[metadata identifier]

Please enjoy!
